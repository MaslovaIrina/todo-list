# To-Do List (учебный проект)

Простое веб-приложение To-Do List на `HTML`, `CSS`, `JavaScript` и `Python` (для локального запуска).

## Возможности

- Добавление задачи
- Просмотр списка задач
- Переключение статуса задачи (выполнено / не выполнено)
- Удаление задачи
- Валидация пустого ввода с сообщением об ошибке
- Сохранение задач в `localStorage` после перезагрузки страницы
- Удобные `data-testid` для ручного и авто-тестирования

## Структура

- `index.html` — разметка интерфейса
- `styles.css` — стили приложения
- `script.js` — логика To-Do и работа с `localStorage`
- `app.py` — простой локальный HTTP-сервер

## Как запустить

1. Откройте терминал в папке проекта.
2. Выполните команду:

```bash
python app.py
```

3. Откройте в браузере:

`http://127.0.0.1:8000`

## Проверка вручную

1. Добавьте новую задачу.
2. Отметьте ее как выполненную и верните обратно.
3. Удалите задачу.
4. Попробуйте добавить пустую задачу и проверьте сообщение об ошибке.
5. Перезагрузите страницу и убедитесь, что задачи сохраняются.
