# To-Do List - учебный проект

Небольшое учебное веб-приложение для управления задачами.  
Проект сделан без сложной архитектуры: чистые `HTML/CSS/JavaScript`, локальный запуск через `Python`, автотесты на `Python + Playwright + pytest`.

## Реализованные функции

- Добавление новой задачи
- Отображение списка задач
- Переключение статуса задачи (`Выполнено` / `Вернуть`)
- Удаление задачи
- Валидация пустого ввода с сообщением об ошибке
- Сохранение задач в `localStorage`
- Фильтрация задач: `Все`, `Активные`, `Выполненные`
- Очистка всех выполненных задач одной кнопкой
- Редактирование текста задачи по двойному клику с сохранением
- Drag & Drop для изменения порядка задач с сохранением порядка
- `data-testid` для стабильной ручной и автоматизированной проверки

## Технологии

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`
- `Python 3` (локальный HTTP-сервер)
- `pytest`
- `Playwright (Python)`

## Запуск приложения локально

1. Откройте терминал в папке проекта.
2. Запустите сервер:

```bash
python app.py
```

3. Откройте в браузере:

`http://127.0.0.1:8000`

## Ручная проверка

Для ручного тестирования используйте чек-лист в файле:

- `todo-manual-test-cases.docx`

Минимальный smoke-сценарий:
1. Добавьте 2-3 задачи.
2. Отметьте одну как выполненную, переключите фильтры.
3. Отредактируйте задачу двойным кликом.
4. Перетащите задачу (Drag & Drop).
5. Перезагрузите страницу и проверьте, что данные сохранились.

## Установка зависимостей для автотестов

```bash
python -m pip install -r requirements-test.txt
python -m playwright install chromium
```

## Запуск UI-тестов

```bash
python -m pytest tests/ui -q
```
